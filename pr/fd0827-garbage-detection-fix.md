# FD08.2.7: 垃圾检测修复 + 应用题支持

## 版本
FD08.2.7

## 改动内容

### 1. `_looks_like_garbage` 修复（3 处）

| 修复 | 问题 | 影响 |
|------|------|------|
| 字符集 → 完整短语匹配 | `[后面格式输出不要答案值跟]` 误伤含"格"字的正常答案（如`价格`） | 应用题全灭 |
| 长度限制 120→500 | 自然语言答案（`甲地市场价格应为 100...`）超过 120 字符 | 应用题全灭 |
| `^\d+[.)]` → `^\d+[.)、]\s` | `0.5` 被误判为 `0. item` 列表项 | 小数答案误判 |

### 2. 新增应用题测试集
- `sample_data/1.jsonl`：10 道计算应用题（880-889）
- 涵盖经济学、微积分应用、物理、概率
- 答案形式为自然语句，非纯数值

### 3. 338 题 GT 修正
- `p1=145, p2=75` → `p1=100, p2=60`（原 GT 计算错误）
