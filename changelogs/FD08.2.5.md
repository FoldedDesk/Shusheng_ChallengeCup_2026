# FD08.2.0 → FD08.2.5 更新日志

## 新增功能

### 提取优化
- **regex 优先提取**：`【最终答案】` / `\boxed{}` 正则直达，跳过 LLM 提取调用（95% 题目走此路径）
- **2/2 共识早停**：前两个候选答案一致时跳过第三个候选生成（60-70% 触发）
- **单候选跳过验证**：仅一个有效候选时直接使用，不调验证器
- **垃圾检测**：`_looks_like_garbage` 答案过滤层，排除占位符、英文散文、模板文字污染
- **Fallback 精简**：截断回退从 3 级减为 2 级

### 归一化增强
- `\text{发散}` → `发散`
- `\left` / `\right` / `\big` 等 LaTeX 修饰符剥离
- `\,` / `\;` / `\displaystyle` 清理
- 函数定义前缀剥离（`S(x) = ` / `f(x) = ` 等）

### 验证器改进
- prompt 增加"不要输出思考过程"约束（尖括号占位符保留）
- `_parse_verdict` 输出质量过滤：跳过模板关键词匹配
- CONFIDENCE 范围校验（0-10）

## 参数变更
- `max_tokens`: 8192 → 12288（减少政策截断）
- `LOCAL_MAX_CONCURRENCY`: 8 → 4（主线程限制，仅影响并行度）

## 提示词变更
- `POLICY_PROMPT`: 增加"请全程使用中文推导，不要输出英文思考过程"
- `EXTRACTION_PROMPT`: 增加 TRUNCATED 检测
- `VERIFIER_PROMPT`: 增加思考抑制约束
- 新增 `FALLBACK_POLICY_PROMPT`，移除 `FALLBACK2_POLICY_PROMPT`

## 测试数据
- 新增 `extended.jsonl`：50 题从易到难
- `test1.jsonl`：修正 4/5 道 ground truth
