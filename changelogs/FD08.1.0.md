# FD08.1.0 — 答案提取 + 一致性加权 + 验证器置信度

**日期**: 2026-06-12

## 更新内容

### 答案提取
- 新增 `_extract_answer()` 方法，按优先级从模型输出提取纯答案：
  1. `【最终答案】` 标记
  2. `\boxed{}` LaTeX 标记
  3. `答案：` 或 `最终答案：` 标记
  4. 行末 `= 值` 模式（适配步骤文本中的计算结果）
  5. 数字开头但非步骤号的行
  6. 末行兜底
- 所有匹配从文本末尾向前搜索，适配 Intern-S 模型的 Thinking Process 前置输出
- `final_response` 从完整推理文本改为提取后的纯答案

### 验证器置信度
- VERIFIER_PROMPT 从二值判断（A/B）改为 10 级置信度
- 新增 `_parse_verdict()` 方法，解析 `VERDICT: A` + `CONFIDENCE: 8` 格式
- verifier `max_tokens` 从 256 调至 1024，给 Thinking 文本留足空间

### 一致性加权
- 新增 `_normalize_answer()` 方法，统一答案格式用于跨 candidate 比较
- 多个 candidate 推导出相同答案时，每人获得 `(k-1) * 0.15` 一致性加分
- AgentConfig 新增 `consistency_bonus_weight` 参数

### trace 增强
- verifier trace 记录 `parsed_correct` 和 `parsed_confidence`
- select_final_response 列出所有 candidate 分项得分
- 新增 `score_summary` 步骤，一行展示评分对比
