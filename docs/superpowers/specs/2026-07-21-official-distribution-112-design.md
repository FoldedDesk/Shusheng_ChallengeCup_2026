# 112 题官方分布回归集设计

## 目标

创建一个可本地自动评估的 112 道中文数学题 JSONL 回归集，严格覆盖给定的初赛题型分布方向。

## 文件与记录格式

输出文件为 `sample_data/official_distribution_112.jsonl`。每行是一条 JSON 对象，包含：

- `idx`：从 3000 到 3111，连续且唯一。
- `problem`：中文题面，可含 LaTeX 公式；不使用图片或外部上下文。
- `answer`：可直接用于本地比对的明确答案或明确结论。
- `subject`：下表中的精确学科名称。
- `source`：固定为 `official_distribution_112`。

## 题目质量约束

所有题目为原创中文题面，答案唯一或有明确的等价表达。计算题选择可手算或可由标准定理直接得到的结果；证明和简答题的 `answer` 写出待证明的具体结论，不使用“显然”“命题得证”等套话。避免缺失条件、OCR 噪声、依赖图形、未定义符号和依赖外部资料的题目。

## 验证

生成后使用标准库脚本验证：文件恰有 112 行；每行是合法 JSON；必填字段均为非空字符串或合法索引；索引连续唯一；`subject` 计数逐项等于指定分布；题面均不含英文自然语言。
